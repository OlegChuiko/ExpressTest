Створити тест

Після натискання кнопки "Створити тест" ви потрапляєте на сторінку, де вам потрібно ввести назву тесту, вибрати необхідний час для його проходження (якщо встановити 0 то час буде не обмежений) та завантажити файл у форматі .txt. Для завантаження файлу ви можете натиснути на поле і вибрати файл із свого комп'ютера або просто перетягнути його у відповідне поле.

Для створення текстового файлу для тесту, що містить запитання та варіанти відповідей, слід дотримуватися наступних кроків:

  1. Відкривши текстовий редактор, додайте запитання для тесту, розмістивши його на початку рядка(нумерувати запитання не потрібно, система зробить це за вас);
  2. Після запитання пропустіть один рядок та додайте список варіантів відповідей. Кожну відповідь записуйте з нового рядка;
  3. Позначте правильну відповідь символом “+” та пропустіть рядок перед наступним питанням.

У результаті має виглядати так:

Питання?

Відповідь 1

Відповідь 2+

Відповідь 3

Відповідь 4
